I've continously refined my web and mobile development skills over the years. Here's a detailed look at my capabilities:

---

## Web Development

I'm passionate about building robust and efficient web applications. My core technologies for achieving this include:

- **Laravel**:  A powerful PHP framework for crafting elegant and scalable backend systems.
- **CodeIgniter**: A lightweight and versatile PHP framework ideal for rapid development and simpler projects.
- **Next.js**: My go-to React framework for building fast, SEO-friendly, and scalable web applications.
- **React**: The foundation for creating dynamic and interactive user interfaces, ensuring a smooth user experience.
- **Node.js**: A JavaScript runtime environment for building scalable server-side applications.
- **MySQL**: A robust and reliable relational database system for managing application data.
- **HTML5, CSS3 and Javascript**: The fundamental building blocks of web development, ensuring responsive and visually appealing websites.
- **WordPress**: A versatile CMS platform for building and managing websites and blogs efficiently.

---

## Mobile Development

Complementing my web development expertise, I also develop mobile applications using:

- **Kotlin**: My preferred language for Android development, enabling the creation of clean, maintainable, and high-performance apps.
- **Java**: A versatile programming language used for building a wide range of applications, including Android apps and enterprise systems.
- **Flutter**: A cross-platform framework allowing me to build beautiful and performant mobile apps for iOS and Android with a single codebase.
- **Firebase**: A comprehensive backend platform providing services like authentication, databases, and cloud functions.

---

## Tools & Platforms

Every developer faces unique challenges. These are the platforms and tools I use to tackle mine:

- **Git & GitHub**:  I use Git and GitHub for streamlined version control and easy project collaboration.
- **Visual Studio Code**: Visual Studio Code is my preferred code editor, enhanced with extensions for increased productivity.
- **Docker**: Docker provides containerization for consistent and efficient development environments.
- **Postman**: Postman is crucial for API testing and seamless Frontend-Backend integration.
